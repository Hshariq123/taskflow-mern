# TaskFlow MERN

A full-stack task management application built using the MERN stack.

## Tech Stack

### Frontend
- React
- Vite
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication
- JWT Authentication
- bcryptjs

---

## Features

- User Authentication
- Secure Login & Signup
- Create Tasks
- Update Tasks
- Delete Tasks
- Mark Tasks as Completed
- Light/Dark Theme
- Responsive Design
- MongoDB Data Persistence
